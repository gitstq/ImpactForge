# ImpactForge - 智能代码变更影响分析引擎

## 项目定位
一个轻量级、零依赖的终端 CLI 工具，用于智能分析 Git 代码变更的影响范围，生成可视化影响图谱和风险评估报告。

## 核心功能
1. **Git Diff 智能解析** - 自动解析 commit/diff 中的代码变更
2. **影响范围分析** - 基于 AST 分析变更对上下游的影响
3. **风险评估引擎** - 智能评估变更的风险等级（低/中/高/严重）
4. **可视化影响图谱** - 生成 Mermaid/PlantUML 格式的依赖影响图
5. **多格式报告输出** - 支持 JSON/HTML/Markdown/SARIF 格式
6. **TUI 交互界面** - 终端图形化操作界面

## 技术栈
- Python 3.8+
- 零外部依赖（纯标准库）
- Git 命令行工具

## 差异化亮点
- 与现有 ChangeImpact-CLI 不同：更智能的 AI 驱动风险评估
- 与 CodeGraph 系列不同：专注变更影响而非静态分析
- 零依赖设计，开箱即用
